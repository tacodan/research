Trackers

Manage and inspect trackers.

List trackers

Returns trackers visible to the authenticated account.

Authorizations:
bearerAuth
query Parameters
page
integer >= 1
Default: 1

Page number.

perPage
integer [ 1 .. 100 ]
Default: 25

Results per page.

status
string
Enum: "online" "offline" "idle"

Filter trackers by current status.

Responses

Production

https://api.trackerapp.example/v1/trackers

Sandbox

Telemetry

Read tracker events and status data.

List telemetry events

Returns recent telemetry records for a tracker.

Authorizations:
bearerAuth
path Parameters
trackerId
required
string
Example: trk_01hrx9h4v4m4q50v7q5j2af8ab

The unique tracker identifier.

query Parameters
start
string <date-time>

Start of the UTC time window.

end
string <date-time>

End of the UTC time window.

limit
integer [ 1 .. 500 ]
Default: 100

Maximum records to return.

Responses

Commands

Send device commands to a tracker.

Send ping command

Queues a ping command so the tracker reports in as soon as possible.

Authorizations:
bearerAuth
path Parameters
trackerId
required
string
Example: trk_01hrx9h4v4m4q50v7q5j2af8ab

The unique tracker identifier.

Request Body schema: application/json
optional
reason
string <= 120 characters

Optional human-readable reason for audit history.

Responses

Production

https://api.trackerapp.example/v1/trackers/{trackerId}/commands/ping

Sandbox

https://sandbox-api.trackerapp.example/v1/trackers/{trackerId}/commands/ping
