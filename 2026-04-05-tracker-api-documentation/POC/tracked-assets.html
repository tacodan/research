Returns tracked assets assigned to the given Scheduler service address id.

This is a compatibility endpoint and intentionally keeps the legacy POST route.

Get tracked asset by EUID

Returns one tracked asset using trackers.dev_eui.

The server normalizes the incoming EUID by trimming and uppercasing it before lookup. Invalid or unknown values return the same 404 response.

bearerAuth
path Parameters

required
string^[0-9A-Fa-f]{16}$
Example: 24E124128C000001

Tracker DevEUI. The server trims and uppercases before lookup.

Log a tracked asset status update request

Temporary compatibility endpoint for the legacy tracked-assets update call.

For launch, this endpoint accepts the posted status/event payload, logs it server-side for later reconciliation, and returns the current tracked-asset compatibility payload.

It does not update retrieval, lifecycle, geofence, or other tracker business state.

Authorizations:
bearerAuth
path Parameters
id
required
integer >= 1
Example: 101

Local tracker id (trackers.id)

Request Body schema: application/json
required
Any of
id
required
integer >= 1

Must match the route tracker id

status
required
string non-empty

Compatibility status value. Logged as received.

event
string non-empty

Legacy compatibility event value. Logged as received.

longitude
number <double> [ -180 .. 180 ]
latitude
number <double> [ -90 .. 90 ]
photoTaken
boolean
euid
string^[0-9A-Fa-f]{16}$

Optional compatibility EUID. The server normalizes it and requires it to match the resolved tracker when supplied.

employeeId
integer >= 1
userId
integer >= 1

Responses

Production API base path

https://iot.keptcompanies.com/api/tracked-assets/{id}/status
